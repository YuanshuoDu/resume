  \textit{How:} I redesigned a manual deployment pipeline using GitHub Actions,

  \textit{Result:} the team shipped continuously instead of in bi-weekly batches,
  and on-call incidents tied to bad deploys dropped to near zero.

  \textit{What I did:} I chose a ResNet50 transfer-learning CNN precisely because

  I fine-tuned the model on the client's dataset, containerised the inference service,
  and deployed it as a Kubernetes microservice. The system is live today, replacing
  manual survey labour and achieving consistent 95\% classification accuracy.
